# Premium Website - Starter

This is a minimal premium-looking Next.js (App Router) + Tailwind starter.

## How to run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the dev server:
   ```bash
   npm run dev
   ```
3. Build:
   ```bash
   npm run build
   npm start
   ```

## Deploy
- Push to GitHub and connect to Vercel (auto-detects Next.js).
